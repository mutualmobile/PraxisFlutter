# Praxis

This is a Sample Flutter app which uses fetches a random Joke and displays it.

It uses Clean Architecture as shown in this [Library](https://github.com/ShadyBoukhary/flutter_clean_architecture). Instead of adding the library as a pubspec dependedency, it has been added as a project dependency to make changes as requried if necessary. Praxis also takes cues from [this](https://github.com/ShadyBoukhary/Axion-Technologies-HnH). 

The purpose of this Library is to show how to implement Clean architecture on a flutter App. This project can be used as a template for new apps. This project is continually evolving to integrate other libraries and techniques to keep it up to date.
